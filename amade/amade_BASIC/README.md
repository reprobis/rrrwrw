### 코드 활용
#### 템플릿의 이용
```
&{template:AmadeSmallDice}{{subject=제목}}{{name=제목상단 작은이름}}{{effect=설명글 출력}}
```
해당 코드를 이용 시, 시트에 포함된 디자인의 템플릿이 채팅창에 출력 됩니다.
